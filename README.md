#CS 4080 Final Project

1. Copy/pull code via git or download files and open them in VSC
2. Open the workspace in VSC and ensure the terminal directory is to the current file
3. Install ruby on computer if haven't already
4. in the terminal run the following command:

**bundle install**

5. Install libvips: https://www.libvips.org/install.html
For Ubuntu (such as using WSL on Windows):

**sudo apt install libvips**

6. run the code in terminal via

**ruby ./app/main.rb**

7. in browser, copy the following into the search bar:

**http://localhost:4567**
