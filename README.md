# CS4080-Final-Project
## Step 1
Download or copy source files into VSC or other code editing software.
## Step 2
Download Ruby if not already installed.
## Step 3
In the terminal, move the directory to the current workspace and do the following:
**gem install sinatra rackup puma**
## Step 4
Run the code in the terminal:
**ruby ./app/test.rb**
## Step 5
In a web browser, copy the following into the search bar
**http://localhost:4567**
